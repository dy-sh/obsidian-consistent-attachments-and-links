This is an [Obsidian](https://obsidian.md/) vault demonstrating features of the [Consistent Attachments and Links](https://github.com/mnaoumov/obsidian-consistent-attachments-and-links) plugin.

## Opening this vault

- **If you already have the plugin installed**, run the **Consistent Attachments and Links: Open demo vault** command — it downloads and opens the latest demo vault automatically.
- **Otherwise**, download `consistent-attachments-and-links.demo-vault.zip` from the [plugin releases](https://github.com/mnaoumov/obsidian-consistent-attachments-and-links/releases), unzip it, and in Obsidian choose `Manage vaults...` → `Open folder as vault` → select the unzipped folder.

On first open, Obsidian treats the vault as untrusted — choose **Trust author and enable plugins** (or enable Community plugins manually), then open [00 Start](<./00 Start.md>).
